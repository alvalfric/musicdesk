MusicDesk



MusicDesk es una aplicación realizada por Alvaro Jesús Alférez Richarte para la asignatura de AISS, tutorizado por Javier Troya Castilla.

Esta aplicación integra las siguientes APIs:

Spotify: Utilizada como motor de busqueda y para metadata de canciones, artistas, grupos, etc...
Eventful: Busqueda de eventos que filtra por artista.
Musixmatch: Devuelve la letra de las canciones buscadas.
Youtube: Permite ver el videoclip o el video relacionado con la canción y comentar sobre ella.
